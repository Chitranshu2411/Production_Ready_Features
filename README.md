# 🚀 Prod Ready Features - Spring Boot Project

## 📌 Overview

This project is a **production-ready Spring Boot REST API** built using clean architecture principles.
It demonstrates how to structure a backend application for scalability, maintainability, and real-world use cases.

---

## 🛠️ Tech Stack

* Java 21
* Spring Boot
* Spring Data JPA
* MySQL
* Lombok
* ModelMapper

---

## 🧱 Project Structure

```
├── controller       → REST APIs
├── service          → Business logic
├── repository       → Database access
├── entity           → Database models
├── dto              → Data Transfer Objects
├── advice           → Global Exception Handling
├── config           → App configurations
├── exceptions       → Custom exceptions
```

---

## ✨ Features

* ✅ Clean layered architecture
* ✅ DTO pattern for request/response
* ✅ Global exception handling
* ✅ Custom error response (ApiError)
* ✅ MySQL database integration
* ✅ ModelMapper for object mapping
* ✅ RESTful API design
